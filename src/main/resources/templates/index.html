Mapa Comparativo SouthEnd - Heroku App

Platform: HerokuApp auto-build through GIT connection*

*Possible to change to AWS or RedHat's OpenShift

Backend J2EE:

- MAVEN (dependency and library management) + SPRING (build MVC)

- HIKARI DB (JDBC lightweight) + Hibernate (JPA - database consistency and ORM)

- ALL data provided through OData web services. No frontend processing on J2EE!

Frontend SPA javascript+html5**

- Angular templating (free) - check Material Angular for UI

- Visual Studio coding (free)

- Unit test using NG (build/serve) with central web service call

Nice beginner tutorial
